# Code Index — main code blocks per file

Lookup hints below use file paths plus a function/section name (no line numbers; grep/Ctrl-F by name).

## Top-level / build

- [wowaddon.py](../wowaddon.py) — Python build/install tool. `VERSION` constant near top. `BuildTool` class: `__init__` writes per-flavour TOCs, `do_install` (copy dirs into Addons folder), `do_zip` (build CurseForge zip), `do_zip_add_dir` / `do_zip_add_root_dir` (recursive), `git_hash`, `create_toc` (template substitution). `main()` argparse entry. Suffix constants: `SUFFIX_CLASSIC`/`-BCC`/`-WOTLKC`/`-Cata`. UI version constants per expansion.
- [toc_template.toc](../toc_template.toc) — TOC template with `${UI_VERSION}` / `${VERSION}` / `${ADDON_TITLE}` placeholders. Authoritative load order — first `embeds.xml`, then `Src/EarlyBoot.lua`, `Src/KvLib/`, `Src/Buffomat.lua`, `Src/Throttle.lua`, then `Src/Const/`, `Src/Knowledge/Spells*` and `Src/Knowledge/Consumes*`, then `Src/Core/`, `Src/Task/`, `Src/Cache/`, `Src/Ui/`, then `Src/Events.lua`, `Src/SpellSetup.lua`, `Src/BuffChecks.lua`, then `Src/Languages/`, then `Src/Toolbox/`.
- [embeds.xml](../embeds.xml) — Loads Ace3 libs in order: `LibStub`, LibDBIcon stack (CallbackHandler, LibDataBroker, LibDBIcon-1.0), CallbackHandler, AceAddon-3.0, AceGUI-3.0, AceTimer, AceEvent, AceConsole, AceConfig, LibUIDropDownMenu.

## Bootstrap / addon entry

- [Src/EarlyBoot.lua](../Src/EarlyBoot.lua) — Single-purpose: `LibStub:NewLibrary("Buffomat-…", 1)` calls registering every module name. Always edit here when adding a new module.
- [Src/Buffomat.lua](../Src/Buffomat.lua) — Main addon file. Module wiring (LibStub locals at top), `BomAddon` class type spec with all dynamic state fields, `BuffomatAddon = LibStub("AceAddon-3.0"):NewAddon(...)`. Functions: `BuffomatAddon:Debug`, `buffomatModule:Color`, `buffomatModule.OptionsUpdate`, `buffomatModule:OptionsInit`, `buffomatModule.ChooseProfile`, `buffomatModule:UseProfile`, `buffomatModule:UpdateBuffTabText` (group label "G1-3,5"), `buffomatModule:ScheduleUpdateTimer`, `buffomatModule:InitUI` (registers minimap LDB), `buffomatModule:InitGlobalStates` (saved-vars init + profile migration), `BuffomatAddon:MakeSlashCommand` (slash table), `BuffomatAddon:Init` (real entry — translations, const, allBuffs, macro, slash, UI), `RegisterBindings`, AceAddon handlers `OnInitialize` / `OnEnable` / `OnDisable`, `buffomatModule:OnFailedCast_LearnedDownrank` (downrank learning from UI errors), hyperlink handlers `EnterHyperlink` / `LeaveHyperlink` / `ClickHyperlink`.
- [Src/Throttle.lua](../Src/Throttle.lua) — `ThrottleModule` rate limiter. `throttleState` table (timestamps, fps, hardware tier). `FastUpdateTimer` resets timestamp. `UpdateTimer` runs every frame: combat / loading-screen guard, profile pick, cooldown checks, modifier-key rescan, `WindowCommand`, `Update`. `Update`, `ClearForceUpdate`, `RequestTaskRescan(reason)` accumulate reasons in `taskRescanRequestedBy`.
- [Src/Events.lua](../Src/Events.lua) — `EventsModule`. Event group constants (`EVT_COMBAT_START/STOP`, `EVT_LOADING_SCREEN_*`, `EVT_BAG_CHANGED`, `EVT_PARTY_CHANGED`, `EVT_SPELLBOOK_CHANGED`, `GENERIC_UPDATE_EVENTS`, error-message lists `ERR_NOT_STANDING/IS_MOUNTED/IS_SHAPESHIFT`). Local handlers `Event_TAXIMAP_OPENED`, `Event_UNIT_POWER_UPDATE`, `Event_PLAYER_*_MOVING`, `Event_CombatStart/Stop`, `Event_Loading*`, `Event_PLAYER_TARGET_CHANGED`, `Event_COMBAT_LOG_EVENT_UNFILTERED` (combat-log buff tracking with `partyCheckMask`), `Event_UI_ERROR_MESSAGE` (auto-stand/dismount/cancelform), `Event_PartyChanged`, `Event_TALENT_GROUP_CHANGED`, spellcast start/stop/channel handlers, `Event_SpellsChanged`, `Event_Bag`.
